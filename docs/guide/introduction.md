# Introduction

## What is Paisa?

**Paisa** (meaning "money" in Hindi/Urdu) is an open-source, cross-platform personal finance and expense tracking application built with Flutter. It provides a polished, feature-rich experience for managing your personal finances entirely on-device — no account, no backend, no internet connection required.

This repository is the **open-source release of v6.0.8**, the last public version before the project was closed. It serves as both a functional app and a reference implementation of modern Flutter best practices.

## Why Paisa?

| Feature | Paisa |
|---------|-------|
| Offline-first | All data stored locally in Hive |
| No account needed | No sign-in, no cloud |
| Multi-platform | Android, iOS, Web, Windows, macOS |
| Open source | GPLv3 licensed |
| Modern Flutter patterns | BLoC, Clean Architecture, GetIt DI |
| Material Design 3 | Dynamic theming (Material You) |

## Project Origins

Paisa was originally developed by **Hemanth S** (`dev.hemanths.paisa`) as a production Android/iOS app. The project was made open source at v6.0.8 before the developer transitioned to a closed-source model. This fork preserves that codebase for community use and contribution.
